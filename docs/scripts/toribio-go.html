Script toribio-go

Toribio application.

This application starts the different tasks and device loaders. It is controlled trough a configuration file (default toribio-go.conf).

Usage:

    lua toribio-go.lua [-h] [-d] [-c conffile|'none'] 
		-h Print help
		-c Use given configuration file (or none). 
		   Defaults to 'toribio-go.conf'
		-d NONE|ERROR|WARNING|INFO|DETAIL|DEBUG|ALL
