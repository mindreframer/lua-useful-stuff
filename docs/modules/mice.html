Module mice

Library for accesing a mouse.

This library allows to read data from a mouse, such as it's coordinates and button presses. The device will be named "mice", module "mice".

Functions

init (conf)
Initialize and starts the module. This is called automatically by toribio if the load attribute for the module in the configuration file is set to true.

Parameters:

  • conf: the configuration table (see conf).
get_pos ()
Get mouse position.

Returns:

    a pair of x, y coordinates.
reset_pos (newx, newy)
Reset position. Fixes the coordinates associated to the current position.

Parameters:

  • newx: number to set as x coordinate of the cursos (defaults to 0)
  • newy: number to set as y coordinate of the cursos (defaults to 0)
set_pause (pause)
Pause the event generation. While the device is paused, no events are generated, nor movements tracked.

Parameters:

  • pause: mode, true to pause, false to unpause

Tables

events
Events emitted by this device. Button presses have single parameter: true on press, false on release.

Fields:

  • leftbutton: Left button click.
  • rightbutton: Right button click.
  • middlebutton: Middle button click.
  • move: Mouse moved, first parameter x, second parameter y coordinates.
conf
Configuration Table. When the start is done automatically (trough configuration), this table points to the modules section in the global configuration table loaded from the configuration file.

Fields:

  • load: whether toribio should start this module automatically at startup.
  • filename: the device file for themouse (defaults to ''/dev/input/mice'').
