Module haar

Library for accesing Haar clasificator.

The device will be named "haar", module "haar", and will generate signals from a external program using OpenCV (see deviceloaders/haar/haar_stream.py). The external haar_stream.py must be started separately.

init (conf)
Initialize and starts the module. This is called automatically by toribio if the load attribute for the module in the configuration file is set to true.

Parameters:

  • conf: the configuration table (see conf).

Tables

events
Events emitted by this device.

Fields:

  • match: a match has been received. Parameters are centerx, centery, sizex, sizey of the matching area.
conf
Configuration Table. This table is populated by toribio from the configuration file.

Fields:

  • load: whether toribio should start this module automatically at startup.
  • ip: where listen for the haar_stream.py data (defaults to '127.0.0.1')
  • port: where listen for the haar_stream.py data (defaults to 45454)
